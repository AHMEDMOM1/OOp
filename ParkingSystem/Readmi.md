# Parking Management System (C++) 🚗🅿️

A professional, object-oriented Parking Management System implemented in C++. This project follows a clean 3-layer architecture (UI, Service, and Models) and features a vibrant, colored console interface.

## 🌟 Key Features
- **3-Layer Architecture**: Separation of concerns between UI, Business Logic, and Data Entities.
- **Smart Parking Logic**: Automated spot allocation and state management.
- **Real-time Duration Tracking**: Accurate calculation of parking duration (Days, Hours, Minutes, Seconds).
- **Interactive Colored UI**: A professional terminal interface using Windows API for a premium feel.
- **Transaction History**: Keeps track of active tickets and archived exit records.

## 🛠️ Project Structure
- `Models/`: Data containers (`Car`, `ParkingSpot`, `ParkingRecord`).
- `Service/`: Business logic and orchestrator (`ParkingSystem`).
- `UI/`: Interactive console screens and visualization (`ParkingScreen`).
- `UseFullLib/`: Helper utilities for time and string formatting.

## 🚀 How to Run
1. Clone the repository.
2. Open `DataStructure.sln` in Visual Studio 2022.
3. Build and run (Ctrl+F5).
