# 🏠 Smart Home Management System

## 🌟 Overview
The **Smart Home Management System** is a professional, interactive C++ console application that allows users to seamlessly manage rooms and a variety of smart devices within their home. It is built strictly using Object-Oriented Programming (OOP) principles, demonstrating robust memory management, dynamic polymorphism, and advanced terminal UI techniques.

## ✨ Key Features

- **Dynamic Room Management**: Add, delete, and list rooms. Each room tracks its maximum device capacity and current usage.
- **Polymorphic Smart Devices**:
  - 💡 **Smart Light**: Control state (On/Off) and precise brightness levels.
  - ❄️ **Smart Air Conditioner**: Adjust cooling temperatures.
  - 🔒 **Smart Door Lock**: Securely lock or unlock.
- **Advanced Console UI**:
  - Uses 256-color ANSI escape codes for a vibrant, modern look.
  - Incorporates Unicode box drawing characters to create elegant tables, menus, and visual capacity bars (`[████░░░░░░░░░░░░░░░░] 1/5`).
  - Professional error handling and status feedback.
- **Robust Memory Management**: Safely handles dynamic allocations and prevents memory leaks using proper constructors, virtual destructors, and pointer tracking.

## 🛠️ Technical Highlights
- **Language**: C++20
- **Design Pattern**: Object-Oriented Design (Inheritance, Polymorphism)
- **UI Mechanism**: Windows Virtual Terminal Processing for ANSI colors and UTF-8 encoding.
